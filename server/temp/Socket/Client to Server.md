### Send Direct Message

- **Endpoint** : /app/chat.direct-message
- **Payload** :

```json
{
    "senderId": 1,
    "chatId": "1_2_1",
    "message": "Hello, World!"
}
```

---

### Send Group Message

- **Endpoint** : /app/chat.group-message
- **Payload** :

```json
{
    "senderId": 1,
    "chatId": "5",
    "message": "Hello all!!!"
}
```
